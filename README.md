# Flutter Practice
Flutter 프레임워크와 Dart 언어에 친해지기 위한 연습장입니다. Flutter와 친해지기 위해 여러 간단한 프로젝트를 접하고 위젯, 생명 주기, 다트 문법 등 다양한 시도를 통해 익히고 있습니다.  

## BMI calculator
- BMI Calculator는 사용자가 작성한 키와 몸무게를 기준으로 계산하여 BMI 수치를 비교하고 비만, 과체중, 저체중 등 현재 상태를 알려줍니다.

## My gallery
- My gallery는 ImagePicker 라이브러리를 활용하여 사용자 앨범에서 사진을 받고 화면에 출력합니다.
- 이미지는 pickMultiImage를 통해 여러개의 이미지를 받을 수 있도록 하였고, PageView 위젯으로 슬라이드 이미지를 구현하였습니다.

## My web browser
- My web browser는 앱 내에서 웹 화면을 띄우는 방법을 연습한 프로젝트입니다.
- 앱 화면을 띄우기 위해 webview_flutter 라이브러리를 사용하였습니다.
- 앱 화면 내에서 웹 화면을 네트워크 문제 없이 보여주기 위해 Manifest에 <uses-permission android:name="android.permission.INTERNET"/> 태그를 추가하였습니다.
- 앱 리스트를 보여주기 위해 AppBar에서 지원하는 속성으로 actions에 버튼을 담아 다른 웹 링크로도 이동할 수 있도록 처리하였습니다.
- 화면에서 앱 이동 기록에 따라 뒤로가기가 원치 않은 방향으로 구현되는 현상이 있어 PopScope를 통해 뒤로가기를 제어하였습니다.

## Stop watch
- Stop watch는 Timer 객체를 활용하여 사용자가 타이머를 시작하고 랩을 기록할 수 있도록 처리하였습니다.
- Stop watch의 기본적인 UI 스타일은 iOS에 Stop watch 어플을 참고하여 만들어 보았습니다.
- Stop watch의 타이머가 화면이 닫히는 경우에는 메모리에서 해제되도록 처리하였습니다.
- 시작, 일시정지, 초기화, 랩 타임 기록 등 버튼 구현을 하였습니다.

## Tilt Sensor
- Tilt Sensor는 sensors_plus 라이브러리를 통해 핸드폰 방향에 따라 가속도 센서 값을 흭득하여 흭득한 값으로 수평에 위치하는 지 시각적으로 보여주는 앱입니다.
- 가속도 센서 값을 가지고 핸드폰이 바닥에 수평으로 놓여져있다면 Stack 위젯으로 이루어진 원형이 가운데로 위치하게 됩니다.

## Xylophone
- Xylophone 앱이며 실로폰 소리를 도, 레, 미, 파, 솔, 라, 시, 도 순으로 위젯을 배치하여 재생할 수 있도록 하였습니다.
- 각 음은 각각 위치한 실로폰 버튼을 클릭하면 소리를 재생합니다.
- 소리 재생에 사용된 라이브러리는 just_audio 라이브러리이며 비동기 처리 등과 같이 적절한 데이터 가공을 통해 소리 데이터를 전달합니다.

## To-Do list
- To-Do list 앱은 할 일 목록을 작성하고 관리하는 앱입니다.
- 간단하게 연습 용으로 만들어졌으며, Hive 라이브러리를 통해 로컬 DB를 활용하였습니다.
- 로컬 DB에 작성한 할 일을 등록하고, 완료, 삭제 등 간단하게 필요한 내용만 구현하였습니다.
